| 내용                        | 해시 후 충돌 해소 | base 62 변환 |
| --------------------------- | ----------------- | ------------ |
| 단축 URL의 길이             | 고정됨            | 가변적       |
| 유일성이 보장되는 ID 생성기 | 필요없음          | 필요         |
| 해시 충돌가능성             | o(해소전략 필요)  | x            |
| 다음 URL?                   | 추측 불가능       | 추측 가능    |
